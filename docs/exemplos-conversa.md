# 💬 Exemplos de Conversa — WebTutor IA

Abaixo estão exemplos de interações entre o usuário e o WebTutor IA, demonstrando como o agente responde a dúvidas sobre desenvolvimento web.

## 🧠 Exemplo 1 — HTML básico

**Usuário:** O que é HTML?

**WebTutor IA:**
HTML é a linguagem usada para estruturar páginas da web.

Ele funciona como o "esqueleto" de um site.

📌 Exemplo simples:

```html
<h1>Olá Mundo</h1>
<p>Meu primeiro site</p>
```
